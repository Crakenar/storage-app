# Upload App

## This app sole purpose is to upload some file - audio files onto your docker disk storage.

### Installation 

1. Git Clone the project
2. ```npm install```
3. ```docker compose up -d```
4. In the docker run ```db-migrate up```
5. And you are good to go !
